js.io demos

by Orbited

  • Chat: A chat demo that connects to an IRC server. By default it will connect to #orbited on irc.freenode.org, but you can change the source so it connects to any IRC server.
  • Telnet: A simple interface for connecting to arbitrary remote servers and sending/receiving lines.
  • STOMP: If you start a STOMP broker on your localhost, this demo will allow you to connect and use its publish/subscribe capabilities, as well as use transactions if the broker supports them. Orbited's Configuration page has instructions for enabling its bundled copy of MorbidQ. Note: MorbidQ does not support transactions.
  • XMPP/Jabber: A minimal XMPP client that connects to a Jabber server running on your local machine.
  • MICS: A barebones MICS protocol-speaking chess client.
  • WebSocket: a simple demonstration of js.io's WebSocket capabilities.
